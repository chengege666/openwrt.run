# Partexp .run

- 来源：`https://github.com/sirpdboy/luci-app-partexp/releases`
- 包含软件包：`luci-app-partexp`
- 支持架构：`all`（与架构无关）
- 运行要求：OpenWrt 或 ImmortalWrt `22.03+`

## 功能

- 一键自动分区格式化
- 分区扩容与自动挂载
- 专为 OpenWRT 设计

## 使用方法

将生成的 `.run` 文件上传到 `/tmp` 并执行：

```sh
chmod +x /tmp/partexp-*.run
/tmp/partexp-*.run
```
