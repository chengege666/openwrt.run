# PassWall2 .run

- Source repo: [Openwrt-Passwall/openwrt-passwall2](https://github.com/Openwrt-Passwall/openwrt-passwall2)
- Included packages: `luci-app-passwall2`, `luci-i18n-passwall2-zh-cn` + full dependency set from upstream release
- Supported architectures: `x86_64`, `aarch64_cortex-a53`, `aarch64_generic`
- Runtime requirements: OpenWrt or ImmortalWrt `23.05+`

## Usage

Upload the generated `.run` file to `/tmp` and execute:

```sh
chmod +x /tmp/luci-app-passwall2-*.run
/tmp/luci-app-passwall2-*.run
```
