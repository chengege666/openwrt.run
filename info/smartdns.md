# SmartDNS .run

- Source: `https://github.com/pymumu/smartdns/releases`
- Included packages: `luci-app-smartdns` and `smartdns`
- Supported architectures: `x86_64`, `aarch64_generic`, `aarch64_cortex-a53`
- Runtime requirements: OpenWrt or ImmortalWrt `22.03+`

## Usage

Upload the generated `.run` file to `/tmp` and execute:

```sh
chmod +x /tmp/luci-app-smartdns-*.run
/tmp/luci-app-smartdns-*.run
```
